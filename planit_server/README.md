# PlanIt

### Install elixir
```
brew install elixir
```

### Install Hex
```
mix local.hex
```
### Install phoenix
```
mix archive.install https://github.com/phoenixframework/phoenix/releases/download/v0.13.1/phoenix_new-0.13.1.ez
```

### Install deps
```
mix deps.get
```

### Run server
```
mix phx.server
```
